# Griddy

A powerful window management tool for macOS, inspired by FancyZones from Windows.

## What is Griddy?

Griddy lets you organize your desktop with custom zones. Drag windows into zones for instant, precise positioning. Create layouts that match your workflow - whether you're coding, designing, or multitasking.

## Features

- **Custom Zones** - Create any layout you want with the visual editor
- **Drag-to-Snap** - Hold a modifier key while dragging to see zones and snap windows
- **Multi-Monitor Support** - Different layouts for each display
- **Keyboard Shortcuts** - Quick layout switching and window movement
- **Lightweight** - Minimal resource usage, runs quietly in your menu bar

## Requirements

- macOS 15.0 (Sequoia) or later
- Apple Silicon or Intel Mac

## Installation

1. Download the latest `.zip` file from [Releases](https://github.com/ChaseIngersol/griddy-releases/releases)
2. Extract and drag `Griddy.app` to your Applications folder
3. Launch Griddy - it will appear in your menu bar
4. Grant Accessibility permissions when prompted (required for window management)

## Auto-Updates

Griddy automatically checks for updates daily. You can also check manually via the menu bar icon → "Check for Updates".

## Getting Started

1. Click the Griddy icon in your menu bar
2. Open the **Layout Editor** to create your zones
3. Hold **⌘ Command** while dragging a window to see zones and snap
